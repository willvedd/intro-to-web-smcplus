Course Objectives

At the end of this course, students should understand basic principles of web design and development, including the knowledge and skills required to edit existing websites as well as create new ones.

Course Description

This course will introduce students to basic concepts and practices in web design and development. Students will learn how to effectively plan, prepare and structure a website based on current best-practices. Students will then learn to build a website with HTML and style their sites using CSS.

At the end of this course, students will be able to:

  • Plan the architecture of a website
  • Structure a website document with HTML
  • Adjust the visual styles of a web document using CSS
  • Gain an overview of the design and development process
  • Edit any existing website

This course is an asynchronous course. Students will watch video lectures on their own time and at their own pace. Each lecture will conclude with a series of exercises designed to test/implement the understanding/knowledge gained during the lecture. Students will then be required to participate in one-hour synchronous web meetings with the instructor. During the synchronous web meetings, students and the instructor will discuss the principles and theories discussed in the video lecture and review the lecture’s exercises.
