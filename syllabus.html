Course Objectives

At the end of this course, students should understand basic principles of web design and development, including the knowledge and skills required to edit existing websites as well as create new ones.

Course Description

This course will introduce students to basic concepts and practices in web design and development. Students will learn how to effectively plan, prepare and structure a website based on current best-practices. Students will then learn to build a website with HTML and style their sites using CSS.

At the end of this course, students will be able to:

  • Plan the architecture of a website
  • Structure a website document with HTML
  • Adjust the visual styles of a web document using CSS
  • Gain an overview of the design and development process
  • Edit any existing website

This course is an asynchronous course. Students will watch video lectures on their own time and at their own pace. Each lecture will conclude with a series of exercises designed to test/implement the understanding/knowledge gained during the lecture. Students will then be required to participate in one-hour synchronous web meetings with the instructor. During the synchronous web meetings, students and the instructor will discuss the principles and theories discussed in the video lecture and review the lecture’s exercises.

Culminating Project

This course will feature a culminating project due at the end of the five weeks. Students are to create a webpage. The subject matter of the page could be anything but students are ecnouraged to focus on something useful like a resume or portfolio.

A successful project will contain

  • An outline of the information architecture of the page
  • Simple sketch or wireframe laying out
  • A valid and semantic HTML document composed of at least 7 different tags
  • An external CSS document with basic styles
  • A media query demonstrating an understanding of responsive web practices
  • Hosted on you github

Schedule

Week 1 - Introduction to the Web

Lecture

What makes a good website, how thoughtful planning can greatly assist the design and devlopment process, looking into website's information architecture, the 10,000ft view of the design process, introduction to user-experience (UX), some modern design trends and caveats.

Exercise

Organize the content of your project and create an information architecture. From this, sketch a few different layouts presenting this information. User-experience message board response.

Meeting

Introduction to what HTML is and the tag syntax. Review of basic HTML tags and how they should be applied. Looking at global and specific HTML attributes that can extend functionality. Learn about valid markup, validation techniques and browser forgiveness.

Assigned Work

Complete small HTML challenge. Revise design or information architecture in accordance with feedback.

Week 2 - Continuing with HTML

Lecture

Continued practice with HTML, specifically introducing the topic of writing semantic HTML and the benefits that come along with it. Introduction into different tags. Introducing students with the web inspector and a more in-depth look into what the browsers actually does with your code.

Exercise

Begin drafting the HTML document for your project. Messageboard response

Meeting

Wrapping up discussion of a semantic HTML document.

Assigned Work

Create an external stylesheet and begin applying some styles. Focus more on basic; don't worry about layout and sizing just yet.

Week 3 - Introduction to Styling

Lecture

Introduction to styling through the "inline" style attribute and some common style properties. Abstracting styles from your HTML with cascading style sheets (CSS). Learning to write syntactically correct CSS, class and ID selectors and what the "C" in CSS means.

Exercise

Create an external stylesheet and begin applying some styles. Focus more on basic; don't worry about layout and sizing just yet.

Meeting

Continued practice with CSS. Looking into two of the more powerful CSS properties: display and position. How we can use those two properties to create site layouts. Common techniques and gotchas.

Assigned Work

Continue styling project with more emphasis on layout this time. Messageboard response.

Week 4 - Principles of Responsive Web Design

Lecture

Focusing on Responsive Web Design (RWD) principles, why it's important and why you should learn and adopt it. How to implement RWD into your web project by avoiding certain units, utilizing certain style properties and media queries.

Exercise

Incorporate a single media query and remove all staticly declared style properties that don't adhere to RWD principles.

Meeting

Allowing for catch-up of HTML/CSS, addressing any questions and concerns.

Assigned Work

Continue developing styles for project, begin pushing to server.

Week 5 - Wrap-up and Other Important Takeaways

Lecture

Wrapping up anything and reviewing HTML/CSS. Going over some more general web development things like common front-end frameworks and content management systems (CMS), paid Web Services and advice on how to continue learning and creating beyond the course.
