Install a Text Editor

A text editor is the primary software we'll be using for the entire course. It is the main tool to write and edit HTML,CSS and practically every other coding language.

Although you can use any text editor for the course, I highly recommend using Sublime Text 2. You can download it at the provided link. Just select the approrpirate operating system.

Afterwards take a minute to open up the week 1 files and make some slight modifications to get a feel for the editor. Think of it as a microsoft word for code.

Installing Git

Git is a software used to assist in organizing large projects and their versions. It allows for multiple people to work on the same files and automatically merge the changes from both users into a single document. Today this is one of the most widely used software for all developers. For this class we'll be using it to help publish our work to a free hosting server.

To install, download the installer from their website and follow the directions.

Create a Github Account

Github is an online service used by most developers to organize and share their projects. To be clear: git and github are two different things. Git is the software that keeps a single project organize and github is the service that uses git to share and publish those projects. Create an account at their signup page.
